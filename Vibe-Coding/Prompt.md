## 提示词工程的三层结构

提示词工程的基础可以拆分为三层：**Prompt 结构层、Prompt 工程方法层、Answer Engineering 层**。三层构成一个完整的提示词系统，从输入到输出都有明确的工程闭环。

## 什么是提示词（Prompt）

用于约束模型行为、限定任务范围、控制输出格式。一个 Prompt 本质上是 **任务描述 + 行为规约 + 输出约束** 的组合。

- [I] 优秀的提示词通常具备以下特征：

- 指令明确（Directive）
- 上下文充足（Context）
- 示例有效（Exemplars）
- 格式固定（Format）
- 风格一致（Style）
- 无歧义（Disambiguation）
