# Agent Practice 想法池

> 项目：AI Agent 实践想法（汇总尚未推进的 Agent 构想）
> 来源：整合自 Second Brain（前端代码审核 Agent + 自动日报 Agent）
> 整合日期：2026-06-17
> 状态：均为想法阶段，暂不开始实现

---

## 一、前端代码审核 Agent

辅助审查前端代码，帮助发现潜在问题并提升代码质量。

### 审核方向

- 代码 bug
- 可维护性
- 组件结构
- 状态管理
- 样式和响应式问题
- 交互一致性
- 可访问性

### 推进路径

先整理一份**前端代码审核清单**，再决定是否做成 Agent。

---

## 二、自动日报 Agent

根据当天禅道任务生成日报草稿，用户确认后再提交日报。

### 需要先确认

- 禅道任务数据如何获取
- 日报提交到哪里
- 日报格式是什么
- 是否需要自动提交，还是只生成草稿

### 推进路径

等**知识库问答 Agent** 和 **Inbox 整理流程**稳定后再推进。

> 注：日报和代码审核都涉及外部系统或代码质量标准，后置更稳。

---

## 三、推进优先级（来自 RAG 项目计划）

| 顺序  | Agent          | 理由       |
| --- | -------------- | -------- |
| 1   | 知识库问答 Agent    | 风险最低     |
| 2   | Inbox 整理 Agent | 最贴合当前需求  |
| 3   | 自动日报 Agent     | 涉及外部系统   |
| 4   | 前端代码审核 Agent   | 涉及代码质量标准 |
| 5   | 项目复盘 Agent     | —        |
