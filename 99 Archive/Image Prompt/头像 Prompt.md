```

整体要求：
两张头像都采用“原创毛绒怪兽角色”设定，风格统一，属于同一个世界观和同一种生物设定，适合作为情侣头像。重点是“同类感”和“成对感”，而不是剧情互动。

角色设定：
主角是原创角色“泡泡呆兽”，是一种抽象、可爱、荒诞的毛绒小怪兽。
它拥有：
- 圆润的头部和短短的身体
- 超大白色鼓眼睛，黑色小瞳孔
- 厚厚的橘色或粉橘色嘴唇
- 脸颊有少量深色圆点
- 毛绒玩偶质感，柔软、蓬松、略带低成本玩具感
- 表情一本正经，但带有呆滞、荒诞、怪萌的气质

头像设定：
第一张头像：角色 A，是“泡泡呆兽”的一个版本，颜色为{{蓝色/奶白色/浅粉色/浅黄色}}，搭配一个简单配件，如{{墨镜/塑料袋头套/咖啡杯/小帽子/围巾}}，表现出冷淡、松弛、呆萌、摆烂的状态。
第二张头像：角色 B，是同一种“泡泡呆兽”的另一个版本，保留相同物种特征和整体造型，但颜色或配件略有变化，如{{另一种浅色系配色/不同配件/不同表情}}，整体依然保持同类感和情侣感。

风格要求：
- 两张图必须是同一种原创角色的不同款式
- 不需要互动，不需要对视，不需要递东西
- 更像“同系列情侣头像”
- 真实毛绒玩偶摄影风格，不是插画，不是二次元
- 近距离半身或大头特写
- 方形 1:1 构图，适合作为社交头像
- 主体占据画面 70% 到 85%
- 背景简洁，米白色、浅灰色、沙发、室内墙面均可
- 光线柔和自然，带一点随手拍、表情包感
- 保留一点笨拙、不完美、不协调的幽默感
- 角色要有强识别度，看起来像原创IP

关键词：
abstract cute plush character, original character design, meme aesthetic, soft toy photography, absurd humor, deadpan expression, quirky couple avatars, matching pair, fluffy texture, simple background, original mascot.

禁止：
不要真人，不要动漫风，不要复杂背景，不要文字，不要 logo，不要水印，不要恐怖元素，不要过度甜美梦幻风。
```

# 三类风格头像


```
生成一组两张独立的原创情侣头像，角色之间不需要互动，但必须属于同一个原创IP系列，具有明显的同类感和成对感。

原创角色名称：咕噜团兽。

咕噜团兽是一种完全原创的毛绒小怪兽，外形特征包括：
圆滚滚的梨形身体，没有明显脖子；
两只大小略微不一致的白色凸起眼睛，瞳孔很小；
脸部中央有一张扁平、厚实、略微下垂的软布嘴巴；
头顶生长着三根短小、柔软、向不同方向弯曲的毛绒触角；
脸颊两侧各有两颗不规则深色绒布斑点；
手臂短小，像两根圆润的毛绒软管；
整体造型笨拙、荒诞，但非常可爱。

第一张头像：
一只雾蓝色咕噜团兽，嘴巴为柔和的橘棕色，戴着一顶尺寸过小的透明雨帽，表情严肃、呆滞，正面看向镜头。

第二张头像：
一只奶油白色咕噜团兽，嘴巴为灰粉色，戴着一副略微歪斜的小号黑色墨镜，表情松弛、冷淡，正面看向镜头。

两张头像不需要对视、不需要递东西、不需要出现共同道具。
通过相同的物种结构、五官比例、材质、摄影方式和背景建立情侣感。

视觉风格：
真实毛绒玩偶摄影；
柔软短绒材质清晰可见；
轻微闪光灯随手拍质感；
带一点低成本玩具摄影的笨拙感；
荒诞幽默、怪可爱、网络表情包气质；
主体为近距离胸像或大头特写；
角色占画面约80%；
1:1方形社交头像构图；
背景为温暖的米灰色室内墙面；
两张图使用完全一致的镜头高度、光线、背景和色温。

禁止：
不要真人；
不要现有动画、影视或游戏角色；
不要传统兔子、猫、熊等常规动物造型；
不要二次元；
不要文字、水印、品牌标志；
不要爱心、玫瑰或明显浪漫装饰；
不要复杂背景；
不要让两个角色出现在同一张图中。
```
