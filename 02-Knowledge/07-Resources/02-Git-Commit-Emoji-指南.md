# Git Commit Emoji 指南

> 来源：Write Your Future（Git/git-emoji-guide）
> 整合日期：2026-06-18

---

格式示例：

```bash
🎨 chore(prettier): format codebase
✨ feat(user): add login page
🐛 fix(table): resolve render issue
```

## 完整对照表

| Emoji | Type | 含义 | 示例 |
|---|---|---|---|
| ✨ | feat | 新功能 | ✨ feat(user): add login |
| 🐛 | fix | 修复 bug | 🐛 fix(table): fix empty data |
| 🎨 | style/chore | 代码格式、美化 | 🎨 chore: format code |
| ♻️ | refactor | 重构代码 | ♻️ refactor(api): optimize request |
| ⚡ | perf | 性能优化 | ⚡ perf(table): improve render speed |
| 📝 | docs | 文档更新 | 📝 docs: update README |
| ✅ | test | 测试相关 | ✅ test: add login test |
| 🔥 | remove | 删除代码/文件 | 🔥 remove: delete unused utils |
| 🚑 | hotfix | 紧急修复 | 🚑 fix: resolve production error |
| 🚀 | deploy | 部署/发布 | 🚀 deploy: release v1.2.0 |
| 🎉 | init | 项目初始化 | 🎉 init project |
| 🔒 | security | 安全修复 | 🔒 fix(auth): update token logic |
| 📦 | build | 构建/依赖 | 📦 build: upgrade vite |
| 👷 | ci | CI/CD | 👷 ci: update github actions |
| 💄 | ui | UI 样式优化 | 💄 style: optimize dashboard ui |
| 🌐 | i18n | 国际化 | 🌐 feat: add english locale |
| ⬆️ | upgrade | 升级依赖 | ⬆️ chore: upgrade pnpm |
| ⬇️ | downgrade | 降级依赖 | ⬇️ chore: downgrade axios |
| 🔧 | config | 配置修改 | 🔧 chore: update eslint config |
| 🚨 | lint | 修复 lint 警告 | 🚨 fix: resolve eslint warnings |
| 🎯 | types | 类型优化 | 🎯 refactor(types): improve typing |
| 🧱 | architecture | 架构调整 | 🧱 refactor: split bff layer |
| 🗃️ | database | 数据库相关 | 🗃️ feat: add user table |
| 🐳 | docker | Docker 相关 | 🐳 build: optimize docker image |
| ☸️ | k8s | Kubernetes | ☸️ ci: update deployment yaml |
| 🚚 | rename | 文件移动/重命名 | 🚚 refactor: move hooks |
| 🧪 | experiment | 实验功能 | 🧪 feat: test ai feature |
| 💚 | fix-ci | 修复 CI | 💚 ci: fix build pipeline |
| 🏗️ | structure | 工程结构调整 | 🏗️ refactor: optimize monorepo |
| 📱 | mobile | 移动端适配 | 📱 style: optimize mobile ui |
