# 正则基础

> 来源：Second Brain（00-Inbox/技术学习记录.md）
> 整合日期：2026-06-18

---

> 正则表达式可以理解为一种字符串扫描规则。

## 普通字符匹配

```javascript
/abc/
```

含义：匹配连续的 `abc`。

```javascript
/abc/.test("abc")    // true
/abc/.test("abc123") // true
/abc/.test("ab")     // false
```

| 部分 | 含义 |
|---|---|
| `/` | 正则开始 |
| `abc` | 正则内容 |
| `/` | 正则结束 |

## `.` 点号

表示任意一个字符。

```javascript
/a.c/
```

含义：`a` 和 `c` 中间可以是任意一个字符。

| 匹配 | 不匹配 |
|---|---|
| abc | ac |
| a-c | abbc |

## `*` 星号

表示前面的内容重复 **0 次或多次**。

```javascript
/ab*c/
```

可匹配：`ac`、`abc`、`abbc`、`abbbbc`

## `+` 加号

表示前面的内容出现 **1 次或多次**。

```javascript
/ab+c/
```

可匹配：`abc`、`abbc`、`abbbbc`
不匹配：`ac`

## `?` 问号

表示前面的内容出现 **0 次或 1 次**。

## `^` 开头符

表示必须从字符串**开头**开始匹配。

## `$` 结尾符

表示必须在字符串**末尾**结束。
