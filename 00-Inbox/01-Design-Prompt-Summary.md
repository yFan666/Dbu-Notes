> 固定创作系统 + 可变主题内容 + 语义转译机制 + 元素融合规则 + 风格边界 + 质量验收

的完整创作框架。

它解决的核心问题是：**无论主题怎么变化，作品都不会偏离既定视觉体系，同时主题又能真正进入设计，而不是只替换标题文字。**

# 一、提示词的底层结构

可以将整条提示词压缩为这个公式：

> **最终作品 = 固定风格系统 × 主题语义 × 视觉转译 × 构图控制 × 边界约束 × 验收标准**

| 模块     | 原提示词中的内容            | 作用           |
| ------ | ------------------- | ------------ |
| 固定风格系统 | 红黄蓝白黑、黑线、非对称网格、现代主义 | 保证作品系列感和风格稳定 |
| 可变输入   | 主题、副标题、比例、语言、用途     | 适应不同创作任务     |
| 语义转译   | 从主题中提取视觉隐喻          | 避免只换文字、不换表达  |
| 元素融合   | 字体、色块、网格、隐喻互相咬合     | 避免元素各自独立     |
| 风格边界   | 禁止赛博、卡通、PPT、摄影等     | 防止模型发生风格漂移   |
| 质量验收   | 十项生成前自检             | 将主观审美变成检查标准  |
